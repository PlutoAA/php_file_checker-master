## Guide

Required: **node.js** and **Composer**

1. ```
   composer install
   ```

1. ```
   cp .env.example .env
   ```

Change DB_DATABASE if needed

1. ```
   php artisan key:generate
   ```

1. ```
   php artisan migrate
   ```

1. ```
   npm i
   ```

1. ```
      npm run dev
   ```

for development

1. ```
   php artisan serve
   ```
